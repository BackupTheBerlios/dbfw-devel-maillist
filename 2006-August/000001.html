Hi Mark,
I have modified the file:

dbfw/js/filter.js

I have modified the function select_add_option:

function    select_add_option    (sel, value, descr)
 {
    var opt = document.createElement("OPTION");
    opt.value = value;
    opt.text = descr;
    if ( isIE4 )
     {
        sel.add(opt);
     }
     else
     {
        sel.appendChild(opt);
     }
 }

allowing to execute the searches also from IE.
